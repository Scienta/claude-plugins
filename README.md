# Directory of Claude Code Plugins

First, add the marketplace:

```
/plugin marketplace add scienta/claude-plugins
```

Then install any of the available plugins:

### [tabletest](https://github.com/nchaugen/tabletest-claude-plugin)

```
/plugin install tabletest@scienta-claude-plugins
```

Included skills:
- `/spec-by-example` helps clarify behaviour in specifications using tables of concrete examples
- `/tabletest` helps write @TableTest-style test methods
